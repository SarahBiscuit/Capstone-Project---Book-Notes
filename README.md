HOW TO INSTALL AND SET UP MY PROJECT

# Clone the repo
git clone https://github.com/SarahBiscuit/Capstone-Project---Book-Notes.git

# Cd into the directory
cd Capstone-Project---Book-Notes

# Install all dependencies
npm install express method-override dotenv pg node-fetch axios
npm install --save-dev nodemon

# Run the app with auto-updates (using Nodemon)
nodemon app.js

KEY FEATURES OF MY PROJECT

A simple web application to manage book notes and users, with a search feature for users by first name and surname.  Pulls in book cover images from external API, to create editable and deletable book codes.  Includes options to re-order the cards by user rating and the year that the user read the book in.
